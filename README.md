## :sunrise: Project Overview

### Challenge Requirements

- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page

### Status

Accessibility

- [x] Page is responsive to different screen sizes
- [x] Interactive elements have clear focus indicator
- [x] Statistic contents are easily announced by screen readers

## :stars: Tech Stack and Approach

### Built with

- **HTML5** – Semantic structure
- **Vanilla CSS** – Flex layout
- **BrowserSync** - Easy development server

### Approach

- Mobile-first workflow for better performance on smaller devices
- Accessibility guided by [ARIA Authoring Practices Guide (APG)](https://www.w3.org/WAI/ARIA/apg/).

## :leaves: Local Development

### Prerequisites

Install the following:

- Git (latest version)
- Node.js (latest LTS recommended)
- pnpm (latest version)

### Setup

```
git clone https://github.com/haquanq-frontendmentor/results-summary-component.git
cd results-summary-component
pnpm install
```

### Start Development Server

```
pnpm dev
```
